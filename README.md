# Weather Trend Forecasting

## Overview
This project analyzes global weather data to forecast future weather trends using multiple forecasting models and advanced data science techniques.

## Dataset
Global Weather Repository Dataset from Kaggle:
https://www.kaggle.com/datasets/nelgiriyewithana/global-weather-repository

## Features Implemented

### Data Cleaning & Preprocessing
- Missing value handling
- Outlier treatment
- Data normalization

### Exploratory Data Analysis
- Temperature trends
- Precipitation analysis
- Correlation analysis
- Seasonal decomposition

### Forecasting Models
- Holt-Winters
- ARIMA
- Prophet
- Ensemble Forecasting

### Advanced Analyses
- Anomaly detection
- Climate analysis
- Air quality analysis
- Feature importance
- Spatial analysis

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Statsmodels
- Prophet
- Matplotlib
- Seaborn
- Plotly

## Results
The ensemble forecasting model improved prediction stability by combining multiple forecasting techniques.
